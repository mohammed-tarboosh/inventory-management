# توثيق مشروع Inventory Management

هذا المجلد يحتوي على التوثيق الكامل لتطبيق إدارة الجرد المبني باستخدام React + TanStack Start وSupabase.

ملفات التوثيق الأساسية:

- `QUICKSTART.md` — تثبيت وتشغيل سريع محلياً.
- `ENVIRONMENT.md` — المتغيرات البيئية المطلوبة وشرحها.
- `API.md` — مرجع الـ API الداخلي وserver functions.
- `ARCHITECTURE.md` — نظرة عامة على بنية المشروع وشرح الملفات المهمة.
- `ROUTES.md` — خريطة المسارات (Routes) وملفاتها.
- `COMPONENTS.md` — مكونات رئيسية ومكتبة الـ UI.
- `SUPABASE.md` — تكامل Supabase: client, admin, middleware، وسياسات RLS.
- `DB_MIGRATIONS.md` — تفاصيل ملفات الترحيل (migrations).
- `DEPLOYMENT.md` — خطوات البناء والنشر لمشروع SSR.
- `CONTRIBUTING.md` — تعليمات الإسهام، lint، format.
- `TROUBLESHOOTING.md` — أخطاء شائعة وحلول سريعة.

ابدأ بـ `QUICKSTART.md` لتشغيل المشروع محلياً.

معلومة حالية للبيئة المحلية:

- `project_id` المحلي مضبوط على `inventory-management` (حسب `supabase/config.toml`).
